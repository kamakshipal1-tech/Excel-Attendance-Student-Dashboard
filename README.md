# Student Attendance & Performance Dashboard (Excel)

This project analyzes student attendance and academic performance using Microsoft Excel.

## Project Objective
To track attendance, eligibility and academic performance of students and present insights using an interactive Excel dashboard.

## Dataset
The project uses two sheets:
- Raw_Data – transactional attendance and marks data
- Student_Master – student reference data (branch and year)

Both sheets are joined using Student_ID.

## Features
- Data cleaning and preparation
- Attendance percentage calculation
- Eligibility logic based on attendance
- XLOOKUP to merge student master data
- Pivot tables for analysis
- Interactive dashboard using pivot charts and slicers

## Analysis Performed
- Subject-wise average marks
- Subject-wise average attendance
- Eligible vs Not Eligible student count
- Branch-wise average marks

## Dashboard Filters
- Subject
- Branch
- Year

## Tools Used
- Microsoft Excel

## Key Skills Demonstrated
- XLOOKUP
- Pivot Tables
- Pivot Charts
- Slicers
- Data cleaning and transformation
